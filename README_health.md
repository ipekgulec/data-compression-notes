# Health Dashboard
Health Dashboard is a health monitoring application developed with SwiftUI. The app allows users to store personal health information and track important health metrics through a simple and modern interface.

# Features
Add and manage multiple users
Track heart rate, water intake, blood pressure, and daily steps
Automatic health status evaluation (Normal, Monitor, Risk)
Interactive dashboard with health cards
View all saved users
Simple and user-friendly design
Technologies Used
Swift
SwiftUI
NavigationStack
State Management (@State, @Binding)
Custom Reusable Components

How It Works
Users can create health profiles by entering personal and health-related information. The application displays the data on a dashboard and automatically analyzes heart rate values to determine a health status.
Learning Outcomes
This project demonstrates:
Data modeling with Swift structures
State management in SwiftUI
User input handling
Navigation between views
Reusable UI components
Conditional logic and data visualization
